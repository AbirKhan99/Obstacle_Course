# Unity Obstacle Course  A beginner 3D obstacle course game made in Unity using C# scripting.  Features:  * Player movement and controls * Collision detection * Time-based logic * Moving and rotating obstacles * Camera follow system  This project was built to practice Unity fundamentals, physics, and gameplay programming.
